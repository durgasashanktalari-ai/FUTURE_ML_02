# AI-Based Customer Support Ticket Classification Using NLP and Machine Learning

## Project Overview

This project classifies customer support tickets into different categories using Natural Language Processing (NLP) and Machine Learning techniques.

The system analyzes customer messages and predicts issue categories automatically for faster customer support management.

---

## Problem Statement

Manual ticket sorting in customer support systems consumes time and reduces efficiency. This project automates support ticket categorization using AI-based NLP techniques.

---

## Features

- AI-based ticket classification
- NLP text processing
- TF-IDF vectorization
- Naive Bayes Machine Learning model
- Category visualization
- Confusion matrix analysis
- Automated support ticket prediction

---
